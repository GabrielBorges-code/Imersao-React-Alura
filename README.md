# ⚛️ Imersão React Alura

Imersão React da Alura número 5.

## 🚀 Tecnologias Usadas

* JavaScript
* React
* Next
* Styled Components
* Vercel

## 🖥️ Para executar o projeto localmente

```
yarn install
yarn run dev
```
